Modelo de Simulación

Aquí puedes encontrar información adicional sobre el modelo de simulación: El siguiente modelo de simulación presenta a los trabajadores, simulando la hora de entrada y salida en horario de oficina. Tomando en cuenta que los trabajadores tienen una llegada aleatoria desde las 8:00 y una salida a las 12:00 Además de una segundo entrada a las 14:30 para salir 18:30

Información del prototipo Arduino

Como componentes del modelo Físico del reloj Biométrico ClockIn contamos con el uso de un módulo wifi esp8266 que tiene la funcionalidad de ser un servidor e intermediario entre la base de datos, y las aplicaciones

Computación Móvil y Ubicua Se hace implementación de Flutter, Arduino, Recursos de Azure como el centro IOT Hub, LogicApp y SQLDataBase
